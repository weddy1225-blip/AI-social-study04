# 時空調查局：家鄉修復官 🕰️

這是一個專為國小四年級社會科「家鄉的變遷」單元設計的網頁互動遊戲。


## 🎮 立即體驗
[點擊這裡開始遊戲](https://weddy1225-blip.github.io/AI-social-study04/)

---


## 📖 遊戲背景
阿強是來自 50 年前的人物，因為時空裂縫來到現代，他的記憶正在逐漸消失。玩家扮演「時空修復官」，透過對話幫助阿強找回關於家鄉變遷的記憶。

## 🌟 特色
- **隨機題庫**：內建 30 題關於交通、通訊、建築、生活習慣的題目。
- **隨機挑戰**：每次遊戲隨機抽取 5 題，增加重遊價值。
- **免安裝**：純 HTML/CSS/JavaScript 開發，任何瀏覽器皆可開啟。
- **教學自動化**：內建自動捲動與進度追蹤功能。
